# 🦦 Otter — Планировщик задач

Мобильное приложение-планировщик, разработанное на **Nuxt 3 + Vue 3 + Pinia + Tailwind CSS**.

## 🎨 Дизайн
- Цвета: Зелёный `#21A038`, Белый, Чёрный, Голубой `#007AFF` (аналог СберБанк Онлайн)
- Mobile-first дизайн (max-width: 430px)
- Все данные — mock (без бэкенда)

## 📱 Страницы

| Маршрут | Описание |
|---|---|

## 📂 Структура проекта
```
otter-app/
├── app.vue                      # Корневой компонент
├── nuxt.config.ts               # Конфигурация Nuxt
├── tailwind.config.js           # Конфигурация Tailwind
├── assets/css/main.css          # Глобальные стили
├── data/
│   └── mockData.ts              # Все mock-данные (задачи, пользователь, настройки)
├── stores/
│   ├── auth.ts                  # Аутентификация
│   ├── tasks.ts                 # Задачи (CRUD, группировка)
│   ├── calendar.ts              # Состояние календаря
│   ├── pomodoro.ts              # Таймер помодоро
│   └── settings.ts             # Настройки приложения
├── composables/
│   └── useTaskModal.ts          # Composable для модалки задач
├── layouts/
│   ├── default.vue              # Обёртка (phone-frame)
│   └── app.vue                  # Layout с BottomNav и FAB
├── middleware/
│   └── auth.ts                  # Защита роутов
├── pages/
│   ├── index.vue                # Лендинг
│   ├── login.vue                # Вход
│   ├── register.vue             # Регистрация
│   └── app/
│       ├── index.vue            # Список задач
│       ├── calendar.vue         # Календарь
│       ├── matrix.vue           # Матрица Эйзенхауэра
│       ├── pomodoro.vue         # Помодоро
│       └── settings.vue         # Настройки
└── components/
    ├── layout/BottomNav.vue     # Нижняя навигация
    ├── tasks/
    │   ├── TaskItem.vue         # Элемент задачи (swipe)
    │   ├── TaskGroup.vue        # Группа задач (раскрываемая)
    │   ├── AddTaskModal.vue     # Модалка добавления задачи
    │   └── TaskDetailModal.vue  # Модалка просмотра задачи
    └── settings/Row.vue         # Строка настроек
```

## 🚀 Установка и запуск

```bash
# 1. Перейти в папку проекта
cd otter-app

# 2. Установить зависимости
npm install

# 3. Запустить в режиме разработки
npm run dev

# 4. Открыть в браузере
# http://localhost:3000
```

## 🔑 Функционал

### Задачи
- ✅ Группировка: Просрочено / Сегодня / Завтра / Позже / Без срока / Выполнено
- ✅ Swipe вправо → выполнено, влево → удалить
- ✅ Добавление задачи с датой, временем, длительностью, приоритетом, уведомлением, повтором
- ✅ Поиск задач

### Календарь
- ✅ 4 вида: День / Неделя / Месяц / Год
- ✅ Временная шкала с задачами
- ✅ Сворачивание раннего (00-06) и позднего (21-00) времени
